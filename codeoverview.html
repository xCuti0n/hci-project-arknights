Code Overview

This is a quick overview of the coding and design choices I made for this project.

Home (home.html)

The home page was inspired by a character selection menu acting as a navigation hub. I added interactive sounds and subtle animations like the hover effect and floating particles.

Figures (figures.html)

This one is the figures page, a three part cinematic layout. The left is a timeline, the center shows the name, and the right is a preview of the figure artwork. This page also includes interactive bits and animations.

About (aboutus.html)

The About page uses a centered three pillar layout with two image pillars on the sides and the main content in the middle.

Contact (contactus.html)

Contact is simple and includes a simple sentence with an email address and the same particle background as the figures page for consistency.

Shared styles & scripts

I tried to keep global stuff in css/styles.css. Per-page files override specific pieces. All interactive behavior is in js/main.js.

Quick design notes

  • I split page CSS so the main stylesheet stays tidy, easier to maintain.
  • Background audio may not autoplay on some browsers and requires user interaction to start playback (Thats why I added a start screen for the user to click in the main home page).
